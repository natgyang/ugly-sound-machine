# ☠ UGLY SOUND MACHINE

> collect sounds, ruin sounds, make beautiful garbage.

UGLY SOUND MACHINE is a chaotic browser-based experimental audio playground built with pure JavaScript and the Web Audio API.

It lets you generate ugly, unstable, noisy, cursed sound textures directly in the browser using synthetic sample generation, layered collages, destruction FX, random corruption systems, and experimental sound manipulation tools.

## 🧪 Features

### 🎛 Experimental Sound Engine

- Synthetic sound generation
- Category-based sound libraries
- Procedural noise creation
- Randomized textures

### ☣ FX System

- Distortion
- Echo / Delay
- Reverb
- Pitch Shift
- Bitcrush
- Wobble
- Stutter
- Flanger
- Tape Stop
- Vinyl Crackle
- Overdrive
- Chop Rate

### 🌸 Sound Collage System

- Layer multiple sounds together
- Build chaotic sound structures
- Ruin collages in real time
- Loop playback support

### 🎲 RUIN IT Button

One button.
Total destruction.
Randomizes:

- sounds
- playback speed
- FX values
- corruption level

Perfect for accidental sound design.

### 🖥 Retro Terminal Interface

Inspired by:

- old sampler hardware
- CRT terminals
- damaged cassette culture
- industrial interfaces
- experimental noise aesthetics
